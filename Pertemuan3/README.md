keluaran digital. Sensor DS18B20 ini memiliki tingkat akurasi yang cukup tinggi yaitu 0,5 yang

sumber. Tegangan sumber untuk sensor suhu DS18B20 adalah 3V sampai 5.5V

### C. ALAT DAN BAHAN

2. Kabel Jumper, LED
